===========
SigZyme
===========

``sigzyme`` is a toolkit of mode decomposition methods.
The latest development version can be found `here <https://github.com/dioph/sigzyme>`_.

Installation
============
``sigzyme`` requires ``torch`` and ``numba``.
Installing the most recent stable version of the package is as easy as::

    pip install sigzyme
